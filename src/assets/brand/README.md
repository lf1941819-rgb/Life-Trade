# Brand Assets

Este diretório contém os assets de marca do LIFE Trade.

## Logo Principal

**Arquivo esperado:** `logo.svg` ou `logo.png`

Substitua este arquivo com o logo real do LIFE Trade quando disponível.

### Dimensões sugeridas:
- SVG: Sem limite de tamanho (escalável)
- PNG: Mínimo 512x512px (para PWA)

### Referência de uso:
```typescript
import logo from './logo.svg';
```

Atualmente, o projeto está usando um ícone placeholder (TrendingUp) na tela de login até que o logo real seja fornecido.
